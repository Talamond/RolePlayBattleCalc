ICM Home Page
=========================

## Installation
```
npm install
```

## Run
```
npm start
```

## Test
```
npm test
```

## React Dev Tools
```
// if you want to disable react dev tools, then go to webpack.config.js and set:
__DEVTOOLS__: false
```

## Deploy
```
npm run deploy
```

## Lint
```
npm run lint
```
